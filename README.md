# Atif Qamar Portfolio

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

For GitHub Pages, publish the generated `docs/` folder:

1. Run `npm run build`
2. Push the updated `docs/` folder to `main`
3. In GitHub Pages settings, set Source to `Deploy from a branch`
4. Select branch `main` and folder `/docs`

## Customize

- Update links in `src/App.tsx`
- Replace `public/profile.jpg`
- Replace screenshots in `public/screenshots`
- Replace social cards `public/og-image.svg`
